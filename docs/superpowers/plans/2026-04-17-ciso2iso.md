# CISO2ISO Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a small Windows-friendly C++ CLI that converts the GameCube sparse-map `.ciso` format into `.iso` files.

**Architecture:** The tool keeps parsing and conversion logic in a focused library layer and exposes a thin `main` entrypoint for argument validation and user-facing errors. The parser reads the `0x8000`-byte GameCube CISO metadata region, interprets the 1-byte block map, and reconstructs the full ISO by copying stored blocks and zero-filling omitted ones.

**Tech Stack:** C++17, Visual Studio solution, optional CMake
